# `@massalabs/prettier-config-as`

> MassaLabs's AssemblyScript [Prettier](https://prettier.io) configuration.

## Usage

**Install**:

```bash
npm install --save-dev @massalabs/prettier-config-as
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@massalabs/prettier-config-as"
}
```
